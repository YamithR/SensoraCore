## DESCRIPCIÓN

El proyecto SENSORA_COLOR_CNY constituye un sistema especializado de análisis óptico diseñado específicamente para aplicaciones educativas en optoelectrónica y sistemas de clasificación automática. Este módulo implementa la integración del sensor óptico CNY70 con capacidades avanzadas de análisis de superficie en tiempo real, proporcionando discriminación entre tonos blanco y negro mediante principios de reflexión infrarroja.

El sistema representa una implementación educativa de tecnologías utilizadas en automatización industrial, control de calidad y robótica, donde la detección óptica es fundamental para navegación, clasificación y control de procesos. La arquitectura cliente-servidor TCP optimizada permite streaming de datos analógicos de alta frecuencia, mientras que el sistema de mapeo dinámico proporciona adaptación automática a diferentes condiciones experimentales sin necesidad de calibración manual.

La implementación combina conceptos fundamentales de optoelectrónica con análisis de datos moderno, creando una plataforma educativa que permite la transición natural desde principios básicos de sensores ópticos hasta aplicaciones avanzadas en visión artificial y automatización inteligente, estableciendo una base sólida para el desarrollo de competencias en tecnologías de detección óptica industrial.

## IMPLEMENTACIÓN

### Implementación de Streaming de Datos Ópticos

La implementación de comunicación para el módulo COLOR_CNY requirió desarrollo de un protocolo optimizado para transmisión de datos analógicos de alta frecuencia:

```python
# Protocolo especializado COLOR_CNY con streaming configurable
def _run(self):
    s.sendall(b'MODO:COLOR_CNY\n')
    resp = s.recv(128).decode('utf-8', errors='ignore').strip()
    
    if 'COLOR_CNY_OK' not in resp:
        self.sig_error.emit("[CNY] ESP32 no acepto COLOR_CNY")
        return
    
    # Configuración de streaming con intervalo personalizable
    s.sendall(f'CNY_START:{self.interval_ms}\n'.encode())
    
    # Recepción optimizada para alta frecuencia
    # Formato: SENSOR:CNY70,ADC:<n>,VOLT:<v>
    buf = ""
    while self._running:
        chunk = s.recv(256)
        buf += chunk.decode('utf-8', errors='ignore')
        
        while '\n' in buf:
            line, buf = buf.split('\n', 1)
            parsed_data = self._parse_cny_data(line)
            if parsed_data:
                self.sig_sample.emit(*parsed_data, time.time())
```

### Implementación de Mapeo Dinámico Automático

La implementación del sistema de calibración automática representa una innovación clave para facilitar el uso educativo:

```python
class ColorCNYLogic(QObject):
    def __init__(self, ui_widget, main_window):
        super().__init__(ui_widget)
        self.ui = ui_widget
        self.main = main_window
        
        # Estado de mapeo dinámico
        self.adc_min = 4095  # Inicialización con valores extremos
        self.adc_max = 0
        self.series = {"t": [], "w": [], "b": [], "adc": [], "volt": []}
    
    def _on_sample(self, adc: int, volt: float, ts: float):
        """Procesamiento con mapeo dinámico automático"""
        # Actualización dinámica de rango por sesión
        if adc < self.adc_min:
            self.adc_min = adc
        if adc > self.adc_max:
            self.adc_max = adc
        
        # Cálculo de rango dinámico
        rng = max(1, self.adc_max - self.adc_min)
        
        # Mapeo porcentual basado en extremos detectados
        white_percent = int(round((adc - self.adc_min) * 100.0 / rng))
        white_percent = max(0, min(100, white_percent))
        black_percent = 100 - white_percent
        
        # Actualización de interfaz y almacenamiento
        self._update_display(adc, white_percent, black_percent)
        self._update_time_series(ts, white_percent, black_percent, adc, volt)
```

```python
def setup_matplotlib_visualization(self):
    """Configuración de gráfica científica optimizada"""
    self.figure = Figure(figsize=(5, 3), dpi=100)
    self.canvas = FigureCanvas(self.figure)
    
    # Configuración de ejes con parámetros científicos
    self.ax = self.figure.add_subplot(111)
    self.ax.set_title("CNY70 %Blanco / %Negro")
    self.ax.set_xlabel("Tiempo (s)")
    self.ax.set_ylabel("%")
    
    # Ventana deslizante optimizada y escala fija
    self.window_s = 25.0  # 25 segundos de historial
    self.ax.set_ylim(0, 100)  # Escala porcentual fija
    
    # Líneas con códigos de color científicos
    self.line_w, = self.ax.plot([], [], label="%Blanco", color="#007bff")
    self.line_b, = self.ax.plot([], [], label="%Negro", color="#dc3545")
    self.ax.legend(loc="upper right")

def _update_time_series(self, ts, white, black, adc, volt):
    """Actualización optimizada de series temporales"""
    if self._t0 is None:
        self._t0 = ts
    
    t_rel = ts - self._t0
    
    # Almacenamiento de datos experimentales
    self.series["t"].append(t_rel)
    self.series["w"].append(white)
    self.series["b"].append(black)
    self.series["adc"].append(adc)
    self.series["volt"].append(volt)
    
    # Actualización de visualización
    self.line_w.set_data(self.series["t"], self.series["w"])
    self.line_b.set_data(self.series["t"], self.series["b"])
    
    # Ventana deslizante dinámica
    if t_rel <= self.window_s:
        self.ax.set_xlim(0, self.window_s)
    else:
        self.ax.set_xlim(t_rel - self.window_s, t_rel)
    
    self.canvas.draw_idle()
```

## PLATAFORMA

### Plataforma de Hardware Optoelectrónico

**Sensor Óptico Principal**:
- **Modelo**: CNY70 Opto-interruptor de Reflexión
- **Tecnología**: LED infrarrojo (~950nm) + Fototransistor de silicio
- **Distancia Operativa**: 0.5mm a 5mm para detección óptima
- **Tiempo de Respuesta**: < 15μs para transiciones rápidas
- **Temperatura de Operación**: -25°C a +85°C rango industrial
- **Encapsulado**: DIP-4 estándar para fácil prototipado educativo

**Microcontrolador de Interfaz**:
- **ESP32 DevKit V1**: Procesador dual-core con ADC de 12 bits
- **ADC Configurado**: Pin D25 para entrada analógica de alta resolución
- **Comunicación WiFi**: TCP optimizado para streaming de datos científicos
- **Alimentación**: 3.3V para circuito del sensor y lógica de control

**Circuito de Acondicionamiento Optimizado**:
- **LED Infrarrojo**: Resistencia limitadora 180Ω para 10mA de corriente nominal
- **Fototransistor**: Resistencia de carga 10kΩ para máxima sensibilidad lineal
- **Configuración**: Emisor común con colector a tierra para inversión de señal
- **Rangos de Operación**: 0-4095 ADC equivalente a 0-3.3V analógico

**Especificaciones del Circuito**:
```
CNY70 Sensor de Reflexión:
├── Pin 1 (Ánodo LED): 3V3 + R 180Ω → Corriente 10mA
├── Pin 2 (Cátodo LED): GND directo
├── Pin 3 (Colector): GND + R 10kΩ pull-down
└── Pin 4 (Emisor): ADC D25 para lectura analógica

Principio de Funcionamiento:
• Superficie Blanca: Reflexión alta → Fototransistor saturado → Voltaje alto
• Superficie Negra: Reflexión baja → Fototransistor cortado → Voltaje bajo
• Distancia Óptima: 1-3mm para máximo contraste y linealidad
```
